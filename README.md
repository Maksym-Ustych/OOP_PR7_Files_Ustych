# OOP_PR14_Files_Ustych

Практична робота №7 з дисципліни «Об’єктно-орієнтоване програмування».

Тема: «Файли в стилі C».

У роботі реалізовано:
- роботу з текстовими файлами;
- роботу з бінарними файлами;
- запис та читання даних з файлів;
- використання функцій fopen_s(), fclose(), fprintf(), fscanf_s();
- використання fread(), fwrite(), fseek().

Проєкт виконано у Microsoft Visual Studio мовою C++.
